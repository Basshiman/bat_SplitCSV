分割する前のファイル名
bat_input.csv

■概要
参考元(http://qiita.com/hiyo3@github/items/1b8a2aa983ad704f612e)
のバッチの変更版。

csvファイルをヘッダー(行数指定可) + 指定行数のファイルに変更するバッチ。

ファイル内のデータをファイル名に含めることができるようになりました。


26行目
set nameBefore=
ファイル名で、ファイル内データの前に付けたい文字列

27行目
set nameAfter=
ファイル名で、ファイル内データの後に付けたい文字列

30、31行目
set batDir=%~dp0
バッチファイルが置いてあるディレクトリのパス　C:~~~~~\　という形式(末尾に \ ついてる)
set outDir=output\
出力場所のディレクトリ　(末尾に \ つけて)

99行目
set name=
ファイル名に含めるファイルを選択する。
空白とカンマ区切りで![n]!の中にデータがあるので、
がんばって使いたいデータを探して入れてください。
